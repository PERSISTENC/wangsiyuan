## 闭包是什么?
    闭包是指有权访问另一个函数作用域中得变量得函数
    
    function demo (){
        const name = '王思渊'
        return function (){
            console.log(name)
        }
    }

    通俗点讲, 返回的子函数里面访问父函数的变量，这一个行为称之为闭包
## 闭包的好处和危害?
    作用域： 作用域更加独立，不会影响和污染全局变量，能更好的封装对象
    内存泄漏：因为变量始终被函数所引用，所以私有变量无法被销毁，常驻内存中，增大了内容消耗，容易造成内存泄漏

## 闭包的应用场景
   防抖 节流 计时器 等等.....
